# TPL Journey 

**Experiencing God's Kingdom in Truth, Power, and Love**

TPL Journey is a nonprofit Christian discipleship ministry that equips individuals and communities to experience God's kingdom through a discipleship framework grounded in Truth, Power, and Love.

## Website Structure

The site follows a discipleship journey progression:

1. **The Kingdom** (`/`) - Homepage with kingdom vision
2. **Your Journey** (`/your-journey`) - Personal discipleship resources
3. **Life Together** (`/life-together`) - Community and DMT groups
4. **Leaders** (`/leaders`) - Disciple-maker development
5. **Intensives** (`/intensives`) - Immersive retreat experiences
6. **About** (`/about`) - Ministry information
7. **Contact** (`/contact`) - Get in touch

## Tech Stack

- **Framework**: Next.js 15+ with App Router
- **Styling**: Bootstrap 5 + Custom CSS
- **Theme**: Based on Resonance Elegant design concept
- **Deployment**: Vercel (recommended)

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

Open [http://localhost:3000](http://localhost:3000) to view the site.

## Deployment

The site is configured for easy deployment to Vercel:

1. Connect your GitHub repository to Vercel
2. Vercel will automatically detect Next.js and configure build settings
3. Set up your custom domain (tpljourney.com)
